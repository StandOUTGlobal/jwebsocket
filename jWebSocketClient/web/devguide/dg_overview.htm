jWebSocket Components and Packages

This page gives you an overview about the various jWebSocket packages and their use.

Basically the jWebSocket Server includes a core, various connectors for different types of clients, various listeners to interpret custom messages and process custom commands as well as streams to broadcast information through the WebSocket network.

The jWebSocket Client includes a JavaScript Client for HTML 5 compliant Web Browsers and a Java Client (targeted for Release 1.2) for java desktop and enterprise applications. The infrastructure diagram gives you an detailed overview.
