Build jWebSocket with Eclipse

This section describes how to import and compile the Apache Maven based jWebSocket sources into Eclipse.

The Eclipse IDE is available at http://www.eclipse.org .

  1. Download jWebSocket source from download area with Tortoise Subversion Client and extract or check out into a folder of your choice.
  2. Change to the root folder (e.g. c:\svn\jWebSocket\trunk)
  3. Run mvn clean install in the command shell
  4. Run mvn clean eclipse:m2eclipse in the command shell,
    this will generate the Eclipse specific files.
  5. Import the project into Eclipse.
