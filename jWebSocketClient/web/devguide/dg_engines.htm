jWebSocket Engines

jWebSocket is designed to support multiple low-level engines beside its own one(s). Currently jWebSocket is implemented on simple TCP Sockets. NIO support is targeted for jWebSocket release 2. We also will provide a support for the JBoss Netty Engine. We are planning to support Apache Mina as well as the Jetty Server .

Interface

setPlugInChain

Each jWebSocket engine needs to support a plug-in chain. This method assigns the plug-in chain to the engine.
