jWebSocket Web Client

This section describes how to use the jWebSocket client and start the demo web site. The jWebSocket client is based on pure JavaScript and does not require any browser plug-ins. It also runs from a local file system, i.e. you do not need to install any web server. If you are not sure whether your favorite browser supports WebSockets please refer to our jWebSocket Cross-Browser Compatibility page.

Setup the jWebSocket Client

To run the jWebSocket client on your MS Windows system please....
  1. Create a folder of your choice (e.g. c:\jWebSocket\client).
  2. Download the jWebSocketClient-<version>.zip from the download area and unzip it into this folder.
  3. Start your browser (please refer Cross-Browser-Compatibility).
  4. Type file:///[path/to/client/]index.htm (e.g. file:///c:/jWebSocket/client/index.html) into the URL input field of the browser.
  5. The browser opens the Welcome page of the jWebSocket client. You now can test the chat, fundamentals, streaming, gaming and RPC demos.
