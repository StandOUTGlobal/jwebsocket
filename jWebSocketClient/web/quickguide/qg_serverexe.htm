jWebSocket Server as Windows Executable

This section describes how to use the jWebSocket Server Executable for Windows Systems. This topic will be interessting for you when you are a developer and intend to develop new client software based upon the jWebSocket Server. The executable is a wrapper around the jWebSocketSAServer-<version>.jar including all required libraries and an icon for the desktop. The benefit is that it easily can be started with a double click from the Windows Desktop without any installation or special configuration.

Like all normal desktop applications the jWebSocket Server Executable is terminated when you log off the system. Thus for production systems it is recommended to use either the jWebSocket Service or the jWebSocket Web Application.

Download, Installation and Start

Please follow the following steps to use the jWebSocket Windows Executable:

  1. Download the jWebSocketServerExe-<version>.zip from the download area.
  2. Unpack the .zip archive with the jWebSocketSAServer-<version>.exe into an arbitrary folder on you local harddisk (e.g. c:\program files\jWebSocket).
  3. Double click the jWebSocketSAServer-<version>.exe and the Server will immediately start:

  4. If you like you can easily create a shortcut to the executable on the Windows Desktop or the Windows Shortcut Bar and start it directly from there.
