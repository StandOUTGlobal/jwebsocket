jWebSocket Base Demo

This demo shows how to easily set up a minimum communication w/o any protocol between a jWebSocket client and a jWebSocket server. This example is based on the class org.jWebSocket.connectors.UserConnector which simply echos the text/plain request back to the client. Currently only Google's Chrome Browser from Version 4.0.249 support HTML 5 WebSockets, you can download it here .
